== Description ==
Funny placeholder is a plugin meant to fill empty spaces (e.g. intended for ads) with funny content. It comes with a widget which shows dummy advertisments (funny pictures). It supports multiple sizes (wide skycraper, medium rectangle, leaderboard).

== Installation ==
1. Upload the folder "funny-placeholder" to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Place the widget where appropriate.

== Frequently Asked Questions ==
= What ads are shown? =
Mostly they are simple jokes, some of them present our partners.

= What languages are supported? =
Currently English and Czech are supported.
